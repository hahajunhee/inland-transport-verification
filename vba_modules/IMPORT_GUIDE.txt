  거래관리 시스템  —  VBA 자동설치 안내
══════════════════════════════════════════════════════════

■ 사전 조건 (최초 1회만)
  Excel 옵션 → 보안 센터 → 보안 센터 설정
  → 매크로 설정
  → ☑ "VBA 프로젝트 개체 모델에 대한 액세스 신뢰"  체크 후 확인

■ 설치 방법 (3단계)
────────────────────────────────────────────────────────
  1단계. 거래관리.xlsm 을 Excel로 엽니다
         "매크로 사용"을 허용합니다

  2단계. Alt + F11 로 VBA 편집기를 엽니다
         메뉴 → 삽입 → 모듈 클릭
         INSTALLER.bas 파일 내용을 전체 복사(Ctrl+A → Ctrl+C)
         VBE 편집창에 붙여넣기(Ctrl+V)

  3단계. F5 키를 누릅니다 (또는 실행 메뉴 → 매크로 실행 → InstallAll)
         "설치 완료!" 메시지가 뜨면 완료

■ 설치 후 사용
────────────────────────────────────────────────────────
  Excel을 재시작하면 대시보드 시트에서 시작합니다.

  버튼 설명:
  ┌──────────────┬────────────────────────────────────┐
  │ 거래 입력    │ 새 거래 입력 + 거래명세서 출력     │
  │ 품목 관리    │ 품목 추가/수정/삭제                │
  │ 거래처 관리  │ 거래처 추가/수정/삭제              │
  │ 거래 조회    │ 과거 거래 조회 및 재출력           │
  │ 대시보드 새로고침 │ 차트 및 요약표 갱신          │
  └──────────────┴────────────────────────────────────┘

■ 파일 위치
────────────────────────────────────────────────────────
  거래관리.xlsm    — 메인 파일
  vba_modules\     — VBA 소스 (백업용)
  output\          — PDF 출력 폴더 (자동 생성)
